#  Skill 24 - PWM to Control Power to LED

Author: Nnenna Eze

Date: 2020-10-23
-----

## Summary
In this skill, PWM was used to control the brightness of the LED. The brightness levels range from 0 to 9, with 0 being the lowest and 9 being the highest intensity. The code stays at each level for 250ms and then moves on to the next highest intensity level.
