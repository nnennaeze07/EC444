#  Skill 32 - Wheel Speed

Author: Nnenna Eze

Date: 2020-11-25
-----

## Summary
For this skill, an optical encoder was wired up to the esp, and placed so that it measured the wheel speed through a pulse counter. A black and white wheel was attached to the buggy wheel in order to count pulses, and from there wheel speed was calculated.
